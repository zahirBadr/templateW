About MOULDIFI Return to index

With MOULDIFI's clean and minimalistic design, you can create an elegant and powerful project. It is designed with passion and precision and is based on latest standards and recommendations. MOULDIFI is powered by Bootstrap v3 - the most popular front-end framework.

This documentation is dedicated to the main element of the template. With each version, we will try our best to update it accordingly. But if you have any questions going beyond what is described here or wish to report any issues, please do not hesitate in contacting us.

Main HTML elements Return to index

Theme files have almost unchanging structure consisting of:

  1. .page-container container for all body elements.
  2. .page-sidebar .main-menu container for sidebar navigation menu.
  3. .site-header container for site header.
  4. .main-header main container for header elements.
  5. .main-container .main-content container for entire content on the page
  6. .footer-main main container for footer.

Layout structure Return to index

Page </head> contains Metadata, stylesheets and JS scripts.

<!DOCTYPE html>
<head>

    <!-- Metadata -->	
	<meta charset="utf-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<meta name="viewport" content="width=device-width, initial-scale=1">
	<meta name="description" content="Mouldifi - A fully responsive, HTML5 based admin theme">
	<meta name="keywords" content="Responsive, Web Application, HTML5, admin theme, Admin Template, business, professional, Mouldifi, web design, CSS3">
	<title>Mouldifi Responsive Admin Theme</title>

    <!-- CSS and jQuery -->
	
    <!-- Entypo font stylesheet -->
	<link href="css/entypo.css" rel="stylesheet">
	
    <!-- Font awesome stylesheet -->
	<link href="css/font-awesome.min.css" rel="stylesheet">

    <!-- Bootstrap stylesheet min version -->
	<link href="css/bootstrap.min.css" rel="stylesheet">

    <!-- Mouldifi core stylesheet -->
	<link href="css/mouldifi-core.css" rel="stylesheet">
	
	<link href="css/mouldifi-forms.css" rel="stylesheet">

	<!-- HTML5 shim and Respond.js for IE8 support of HTML5 elements and media queries -->
	<!-- WARNING: Respond.js doesn't work if you view the page via file:// -->
	
	<!--[if lt IE 9]>
	<script src="js/html5shiv.js"></script>
	<script src="js/respond.min.js"></script>
	<![endif]-->

</head>

Right-to-Left Language Support Return to index

Adding support for language written in a Right-To-Left (RTL) direction.

Change the value of lang attribute in html tag as per the language selection. Here is an example, where language_code needs to be replaced with the specific language's code like ar for Arabic.

<html lang="language_code">
...
</html>

Following are two css files you will need to add AFTER all the other css files: 

<link href="css/bootstrap-rtl.min.css" rel="stylesheet">
<link href="css/integral-rtl-core.css" rel="stylesheet">

Bingo! you have RTL support in Integral now
